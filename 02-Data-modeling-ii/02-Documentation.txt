Documentation: ETL Process for JSON Data into Cassandra Database
เอกสารนี้เป็นเอกสารคำอธิบายขั้นตอนการนำเข้าข้อมูล JSON เข้าสู่ฐานข้อมูล Cassandra 
โดยใช้กระบวนการ ETL (Extract, Transform, Load) โดยสกัดข้อมูลจากไฟล์ JSON, แปลงข้อมูลตามความเหมาะสม
และโหลดข้อมูลลงในตาราง Cassandra พร้อมกับอธิบายโมเดลข้อมูลที่ใช้เก็บข้อมูลในฐานข้อมูล 
ด้วยรายละเอียดสั้น ๆ คุณสามารถเข้าใจการทำงานของกระบวนการและโมเดลข้อมูลได้อย่างง่ายดาย 
และนำไปใช้ในการพัฒนาระบบได้อย่างมีประสิทธิภาพและเป็นระบบมากขึ้น
